# CareHero Game TS

Protótipo de app **CareHero** em **React Native + Expo + TypeScript**, com visual inspirado em jogos (Clash Royale / LoL client) e cobrindo as telas pedidas na atividade:

- Login
- Cadastro
- Tela principal (Hall do Herói)
- Missões
- Índice Global
- Squads / Eventos
- Perfil / Personagem

## Como rodar

1. Instalar dependências:

```bash
npm install
```

2. Rodar em modo web (ótimo para banca):

```bash
npm run web
```

3. Ou rodar no fluxo padrão Expo:

```bash
npm run start
```
