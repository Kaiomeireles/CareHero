# 📘 **README – CareHero: Gamificando a Saúde da Humanidade**

## 🦸‍♂️ **CareHero – Transformando saúde em uma jornada épica**

O **CareHero** é um aplicativo gamificado criado para promover hábitos saudáveis de forma envolvente, divertida e **altamente inspirada em jogos modernos**, como:

* Clash Royale
* League of Legends
* MMORPGs
* Sistemas de progressão e temporadas

Nosso objetivo é transformar **saúde**, **qualidade de vida** e **bem-estar** em uma experiência épica — onde cada pessoa se torna um **Herói da Humanidade Saudável**, contribuindo individualmente e coletivamente para elevar o “Índice Global”.

---

# 🧠 **1. Contexto do Projeto**

A atividade solicitada pela FIAP pedia:

✔ Tela de **Login**
✔ Tela de **Cadastro**
✔ Tela principal com visão geral
✔ Elementos de RPG
✔ Alguma forma de incentivo/engajamento
✔ Criatividade aplicada ao tema saúde

Nós decidimos levar isso ao **próximo nível**, transformando a proposta em um **APP de verdade**, funcional, com:

* Navegação completa
* Lógica de XP e níveis
* Missões com categorias
* Sistema de streak
* Índice global colaborativo
* Modo squads
* Estética “game client” moderna

---

# 🎮 **2. Problema**

As soluções de saúde tradicionais sofrem de:

* Falta de engajamento
* Experiência repetitiva
* Pouca motivação contínua
* Usuários abandonando após semanas
* Baixo senso de propósito ou recompensa

Na vida real, cuidar da saúde não tem “feedback instantâneo”, o que reduz adesão.

---

# 🚀 **3. A Solução – CareHero**

Criamos um app que combina:

### **🧬 Saúde + Gamificação + Competição saudável + Propósito coletivo**

Cada usuário é um **Herói**, com:

* **Nível**
* **Liga** (Bronze, Prata, Ouro, Diamante, Lendário)
* **XP**
* **Streak de dias**
* **Retrato/Avatar estilo RPG**
* **Progressão de temporada**

Cada missão concluída melhora:

* **A saúde individual do herói**
* **O Índice Global — pontuação coletiva da humanidade saudável**

---

# 🏹 4. O Storytelling

A humanidade passa por um momento crítico: sedentarismo, estresse, má alimentação.

A CarePlus convoca pessoas para se tornarem **Heróis**, unidos no propósito de elevar a "Pirâmide da Humanidade Saudável".

Cada hábito saudável realizado é uma batalha vencida.
Cada missão concluída eleva o mundo ao próximo estágio.

O app transforma a vida saudável em uma **aventura cooperativa**.

---

# 🧱 **5. Funcionalidades Principais**

### 🟢 **1. Login e Cadastro**

* Fluxo completo
* Campos funcionais
* Validação simples
* Usuário demo para apresentação

---

### 🟢 **2. Hall do Herói (Home)**

Tela principal com:

* Retrato do herói
* Nível
* Liga
* Barra de XP
* Streak
* Progresso da temporada
* Cards de metas e marcos
* Destaque do Índice Global

Visual estilo client de jogo **Clash Royale / LoL**.

---

### 🟢 **3. Sistema de Missões**

Cada missão possui:

* Categoria (Água, Movimento, Mente, Nutrição)
* Dificuldade
* XP
* Descrição
* Botão **Concluir**
* Missões concluídas ficam marcadas

Conclusão gera:

* XP
* Aumento de nível
* Aumento de streak
* Aumento do Índice Global
* Animação/alerta de confirmação

---

### 🟢 **4. Índice Global**

Visão coletiva:

* Pontuação total de todos os heróis
* “Pirâmide da Humanidade Saudável” com estágios
* Barras visuais de progresso
* Explicação do conceito

---

### 🟢 **5. Modo Squads**

Modo social inspirado em:

* Clãs de Clash Royale
* Guildas de MMORPG
* Eventos da vida real (triathlon, corrida, golf)

Permite:

* Entrar em squads
* Ver banners de eventos
* Entrar no Triathlon CareHero
* Jogar missões cooperativas
* Criar lobbies de treino

É a ponte entre **eventos físicos** e a **gamificação do app**.

---

### 🟢 **6. Perfil / Personagem**

Tela com:

* Status geral
* Nível
* Liga
* Streak
* XP
* Função do herói
* Possíveis “slots de talentos” (conceito futuro)

# 📱 **8. Como rodar o projeto**

### ▶️ **Modo Web (recomendado para banca)**

Na pasta do projeto:

```bash
npm install
npm run web
```

Se o macOS bloquear o Metro por “too many files”, usar:

```bash
npx expo export:web
```

E abrir o arquivo:

```
dist/index.html
```
